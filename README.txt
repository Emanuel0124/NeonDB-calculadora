# Declaración de Renta
Este programa está diseñado para calcular el impuesto sobre la renta basado en varios 
parámetros financieros ingresados por el usuario.



## Requisitos Previos

- Software necesario: Python instalado en tu sistema. Puedes descargarlo desde python.org.
Kivy instalado. Puedes instalarlo usando pip: pip install kivy.


## Instrucciones de Ejecución

1. Clona el repositorio: clona el repositorio en tu máquina local con el comando:
 git clone https://github.com/Emanuel0124/Declaraci-n_de_renta.git

2. Accede al directorio del proyecto:
cd Declaraci-n_de_renta

3. Ejecuta el programa:
python declaracion_de_renta.py

4. Sigue las Instrucciones en Pantalla:
El programa te guiará a través de los pasos necesarios para calcular tu declaración de renta.
Introduce la información solicitada según corresponda

5. Visualizar los Resultados:
Una vez completado el proceso, el programa mostrará los resultados de tu declaración de renta
